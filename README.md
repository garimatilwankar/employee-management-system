# Employee Management System

A Full Stack Employee Management System built using React, Node.js, Express.js, PostgreSQL (Neon DB), and REST APIs.

## Features

- Dashboard Statistics
- Department Management
- Skills Management
- Employee Creation
- Employee Listing
- SQL JOIN Queries
- PostgreSQL Cloud Database (Neon)
- React Frontend
- Express Backend

## Tech Stack

- React
- Node.js
- Express.js
- PostgreSQL (Neon)
- Axios
- Multer
- Git & GitHub
